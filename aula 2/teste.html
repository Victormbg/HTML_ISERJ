Redes de Computadores

Uma rede de computadores consiste de dois ou mais computadores conectados entre si. Uma rede de computadores pode ser de três tipos:

Rede de área local, rede de área metropolitana e rede geograficamente distribuída.

Em uma rede é possível compartilhar serviços, dados e impressoras, trocar mensagens, compartilhar acesso a web.

Tipos de Redes

Uma rede de computadores pode ser de três tipos: LAN, MAN e WAN.

-------------------------------------
| Sigla | Descrição                 |
-------------------------------------
| LAN   | Local Area Network        |
| MAN   | Metropolitan Area Network |
| WAN   | Wide Area Network         |
-------------------------------------

Navegadores Web

São programas utilizados para navegar na web e visualizar suas páginas. Os principais navegadores são:

  • Apple Safari
  • Google Chrome
  • Internet Explorer
  • Mozilla Firefox
  • Opera

Editores HTML

São programas utilizados para criação e alteração de páginas web. Os principais editores HTML são:

  1. Adobe Dreamweaver
  2. Komposer
  3. Notepad++
  4. Sublime Text
  5. WYSIWYG Web Builder

Profissionais de Web

Web designer
É o profissional responsável pela construção do layout e das interfaces de interação com os usuários.
Web developer
É o profissional responsável pela construção dos programas que rodam por trás das interfaces das páginas.
